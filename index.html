Visual Code Studio and Console Screens

To set up VS Code, make sure a folder is set up and create a file for html and if needed, css. From there start coding.

The console screen is a useful tool in coding. It allows you to see where possible mistakes are and in what line of coding they are in. To open the console screen, right click in your creation and click inspect. After that, click console and it will show you the outputs of your codes. But, in order for that to work you have to type console.log(...) after the line of code.
